Proposito

Este repositorio contiene un programa desarrollado bajo el paradigma de Programación Funcional, orientado al procesamiento y manipulación de archivos CSV.
El sistema permite gestionar una lista de ciudades junto con sus fechas y temperaturas registradas, ofreciendo las siguientes funcionalidades principales:

- Visualizar una gráfica de barras que muestre las temperaturas de las ciudades seleccionadas por el usuario, dentro de un rango de fechas específico.
- Determinar qué ciudad presentó la temperatura más alta y cuál registró la más baja dentro del mismo intervalo.
